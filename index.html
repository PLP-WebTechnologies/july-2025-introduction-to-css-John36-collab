Understanding the Box Model

Every element in web design is a rectangular box. The CSS Box Model describes how these boxes are sized and spaced.

Margin Box

Outermost layer, creates space between elements.

Border Box

Surrounds the padding elements and content.

Padding Box

Space between the border and content.

Content Box

Where your actual content lives.

CSS Properties to Remember

  • margin: Space outside the border
  • padding: Space inside the border
  • border: The edge around the element
  • width/height: Dimensions of the content
